# Company Efficiency Analysis (N & M Metrics)

Python program for calculating and visualizing cost dynamics and sales efficiency based on historical CSV data. The software analyzes financial indicators of companies from 2008 to 2025.

üìå **Problem Description**
The script reads a database of companies and calculates two key parameters for business financial health assessment:
1. **Parameter N (Cost Increase)**: Reflects the year-on-year change in a company's absolute costs (the difference between revenue and profit).
2. **Parameter M (Efficiency Ratio)**: Shows the ratio of revenue growth to cost increase. It helps understand how much additional revenue is generated by each unit of cost invested.

üßÆ **Mathematical Model**
For each pair of consecutive years ($t-1$ and $t$), the following indicators are calculated:
‚Ä¢ Costs: $Costs = Sales - Profit$
‚Ä¢ X-Axis (N): 
$$N = (Sales_t - Profit_t) - (Sales_{t-1} - Profit_{t-1})$$
‚Ä¢ Y-Axis (M): 
$$M = \frac{Sales_t - Sales_{t-1}}{N}$$

üöÄ **Features**
‚Ä¢ **Flexible Selection**: User enters the company name manually.
‚Ä¢ **Dynamic Range**: Data filtering within the 2008‚Äì2025 period.
‚Ä¢ **Error Handling**: Protection against division by zero and missing data for specific years.
‚Ä¢ **Visualization**: Interactive scatter plot with trend lines and period labels.

1. Clone the repository:
```bash

```
2. Install dependencies:
```bash

```
3. Prepare data:
Place your `export_dataset.csv` file in the folder specified in the `FOLDER_PATH` variable inside the script. The file must have the following columns:
`rank, name, country, sales, profit, asset, market_value, year, id`.
